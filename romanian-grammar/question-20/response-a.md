### Cum se identifică subiectul neexprimat?

Într-o propoziție, subiectul neexprimat (care nu este scris sau rostit) se identifică analizând **forma personală (desinența) a verbului** cu funcție de predicat sau deducându-l din **context** (din propozițiile anterioare). Terminația verbului poartă informația de persoană și număr, ajutându-ne astfel să identificăm cine face acțiunea.

### Care este diferența dintre subiectul inclus și cel subînțeles?

Secretul pentru a face diferența între ele stă în **persoana verbului**.

1. **Subiectul inclus** apare atunci când verbul predicativ este la **persoana I sau a II-a** (singular sau plural). Deoarece terminația verbului la aceste persoane ne arată precis cine efectuează acțiunea (_eu, noi_ pentru persoana I, respectiv _tu, voi_ pentru persoana a II-a), nu este nevoie să exprimăm pronumele. Acesta este practic cuprins (inclus) în forma verbului. În textele noastre îl exprimăm explicit doar atunci când dorim să îl scoatem în evidență sau să punem două subiecte în antiteză.
2. **Subiectul subînțeles** apare atunci când verbul predicativ este la **persoana a III-a** (singular sau plural). În acest caz, subiectul (_el, ea, ei, ele_) este cunoscut din propozițiile anterioare și nu se mai exprimă pentru a se evita o repetiție neplăcută. El este pur și simplu dedus (subînțeles) din context.

### Exemple pentru fiecare situație:

- **Exemplu pentru subiect inclus:**
    - „_Plec la munte._”
      Aici, verbul „plec” este la persoana I, numărul singular. Astfel, prin forma verbului, știm imediat că subiectul este **eu**. Acesta este un subiect inclus.

- **Exemplu pentru subiect subînțeles:**
    - „_În vaduri **ape** repezi curg / Și vuiet dau în cale._”
      În prima propoziție („_În vaduri ape repezi curg_”), subiectul exprimat este **ape**. În a doua propoziție („_și vuiet dau în cale_”), predicatul este verbul la persoana a III-a „dau”, însă subiectul său nu mai este repetat; el se deduce ca fiind **ele** (apele) și poartă denumirea de subiect subînțeles.
